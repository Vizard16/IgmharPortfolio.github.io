Automated piston-oriented assembly system

Omron Mobile Robot, Cognex Vision System, UR Robot

University project carried out with Siemens & Manchester Robotics where we developed an entirely automated control of a manufacturing process of a piston assembly, using a Mobile Robot Omron LD-60 AGV, a HMI for the interaction with the operator, a Cognex camera for the vision system, two colaborative robotic arms for assembly: UR Cobot and Omron Cobot. On this Project I was responsible for the mapping of the Automated Guided Vehicle and programming the trajectories that the robot needed to follow.

    August-October 2023

    Automated Process of a
    Pick and Place using mobile Robot and Robotic Arm

    ROS, Python, Mobile Robotics, PID Control

    On this challenge it was required to make an implementation of a coordinated system of an xArm Robot and a Mobile Robot in which they needed to make a task together. This was programmed with ROS (Robot Operating System) using Python. The task was to perform a pick and place in different positions, the mobile robot will take the arm to the position of pick and place. Crucially, to ensure controlled, stable, and precise movement throughout this operation, the robot employs a PID control closed-loop system, required to make the robot go to the different points in a defined space, and also it was required to make both robots communicate with each other.

    PRECISE TEMPERATURE CONTROL OF TRACTOR CABS

    Programming, Control, Electronics, Arduino, Sensors and Actuators, Digital System

    This project was carried out hand in hand with John Deere (training partner), where we developed the precise control of the cab of a John Deere tractor.     May-June, 2022

    Automatic ELECTROPLATING PROJECT

    Control, Electronics, Arduino, Sensors and Actuators

    Project developed in Arduino and LabVIEW where a galvanizing process had to be done automatically. The operation and logic was programmed using an arduino and the real time metrics of the process were displayed using LabView. The project used a Mosfet for electric control. Step Motor and Servo Motor were used as actuators and limit switches as sensors. At the end this project was tu make a mechatronical integration by making a functional electroplating machine.
